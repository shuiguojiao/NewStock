1) 修改auto.ini文件内容
   FormNumber为申购新股时用的不同券商下单程序的个数
   按照FormNumber个数，设置下面二个参数
   FormName？ 为最后确认画面的标题
   LabelName？为最后确认画面中确认键的名称

2）在打新股时，先启动N个账户下单程序，分别填好股票号码，价格及申购股数，按确认，跳出最后一个确认窗口，
   启动auto.exe文件,先确定N个账户下单程序的最后一个确认窗口是否出现,然后按<开始>键,就可以在最快的时间
   内同时申购新股了.之后一定要在<当日委托>中确认是否申购成功.
   注:<开始>键按下后,会在上面出现N个账户<最后确认画面的标题>和<确认键的名称>,假如没有出现的话请确认auto.ini
   文件的设置.
