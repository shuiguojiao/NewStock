使用说明：
1) 修改auto.ini文件内容（现用的模版在华泰通信达上验证过）
   FormNumber为申购新股时用的不同券商下单程序的个数
   按照FormNumber个数，设置下面二个参数
   FormName？ 为最后确认画面的标题
   LabelName？为最后确认画面中确认键的名称

2）使用步骤
	先启动N个账户下单程序，分别填好股票号码，价格及申购股数，按确认，跳出最后一个确认窗口，
	启动auto.exe文件
  点击<列出>,会列出所有已找到的窗口和确认键。
  点击<打新>，开始申购新股.之后一定要在<当日委托>中确认是否申购成功.
